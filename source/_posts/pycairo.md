# PyCairo

PyCairo 是一个用来与 Cairo 库一起工作的 Python 模块。它是到 Cairo C 库的 Python 绑定的集合。除了一些需要以更加 Pythonic 的方式处理的情况，它与 C API 是紧密匹配。
